# Real Scan Input Guide

Place small local validation samples outside Git. Recommended local folder:

```text
D:\Proyectos de desarrollo de Software\BookRescue\validation-inputs
```

Do not commit copyrighted books, private scans, or large image/PDF files.

Use only samples that are:

- public domain;
- self-owned;
- explicitly permitted for testing;
- cropped to the smallest page or region needed.

Suggested file types:

- `.png`
- `.jpg`
- `.jpeg`
- `.pdf`
- `.tif`
- `.tiff`

Keep generated validation results in an ignored local output folder such as:

```text
D:\Proyectos de desarrollo de Software\BookRescue\validation-output
```

Before sharing results, verify that screenshots, PDFs, DOCX files, and logs do not expose private or copyrighted content.
